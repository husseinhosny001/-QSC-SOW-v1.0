# -QSC-SOW-v1.0
يهدف هذا المشروع إلى تصميم وتصنيع واختبار رقاقة ضوئية كمومية هجينة متكاملة (Quantum Photonic Integrated Circuit - QPIC) تعمل كمعالج تشفير سيادي. ستدمج الرقاقة مصادر فوتونات متشابكة، ودوائر ضوئية خطية/غير خطية لمعالجة الكيوبتات الضوئية، وكواشف فوتونات مفردة، ودوائر تحكم إلكترونية على منصة سيليكونية واحدة.
